# GAN-Synthetic-Data-Generation
MSc coursework on Generative Adversarial Networks for medical imaging, cybersecurity, and creative AI.

## Overview

This repository contains my MSc coursework on Generative Adversarial Networks (GANs).

The assignment investigates the use of GANs for synthetic data generation across three application areas:

- Medical image generation
- Cybersecurity traffic generation using CICIDS 2017
- Creative AI using the Google QuickDraw dataset

## Technologies

- Python
- PyTorch
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- TorchMetrics

## Experiments

### Part 1
Medical image generation using GANs.

### Part 2
Synthetic cybersecurity traffic generation using the CICIDS 2017 dataset.

### Part 3
Creative sketch generation using DCGANs trained on the Google QuickDraw dataset.

## Evaluation

The generated samples were evaluated using:

- GAN training loss
- Principal Component Analysis (PCA)
- Statistical comparison
- Frechet Inception Distance (FID)
